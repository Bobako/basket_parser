Парсер баскетбольных матчей.
Для работы парсера:
1) Установить python3 (https://www.python.org/downloads/windows/). ВАЖНО: в самом начале установки поставить флажок в нижней части окна (добавить питон в паф)
2) Распоковать папку с парсером, зайти в эту папку и в строке, где отображается путь к папке стереть все и написать cmd
В открывшемся терминале вбить команду: pip install -r requirments.txt
Дождаться установки библиотек.
3) Тут же, в терминале, написать python bb_parser.py
Это запустит парсер на парсинг всех событий, начиная с 1 января 2020. Если нужны данные только по новым событиям, укажите желаемую стартовую дату после команды запуска.
Например: python bb_parser.py 09-05-2021                 ВАЖНО: формат указания даты долженг быть именно таким (день-месяц-год)
